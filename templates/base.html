Sobre o Projeto

Marsalis é o primeiro projeto da Opentech em desenvolvimento de software. O objetivo é desenvolver um sistema de helpdesk com ordem de serviço a ser utilizado em ambientes de suporte técnico de rede e suporte de sistemas corporativos, bem como testar o framework de desenvolvimento web Django, para tornar-se sua ferramenta de fato.
